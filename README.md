# Amazon Agent Workspace

用于 OpenClaw 的亚马逊跨境电商多 Agent 工作区。

## 目录
- `IDENTITY.md`：主 Agent 身份
- `SOUL.md`：主 Agent 核心规则
- `USER.md`：用户偏好
- `agents/listing-agent/`：Listing 撰写 Agent
- `agents/ops-agent/`：亚马逊运营 Agent
- `skills/`：工作区级 Skills
- `data/`：预留数据目录

## 建议上传到 Git
直接把本目录推送到你的仓库即可。

## OpenClaw 使用
把 OpenClaw 的默认 workspace 指向本目录，然后重启或新开会话。
