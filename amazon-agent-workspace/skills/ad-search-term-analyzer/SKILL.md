# Ad Search Term Analyzer

Use this skill on ad search term reports.

## Focus metrics
- impressions
- clicks
- ctr
- cpc
- orders
- conversion rate
- spend
- sales
- acos
- roas

## Group terms into:
- **high_click_high_conversion** - keep, increase bid
- **high_click_low_conversion** - optimize
- **low_click_high_conversion** - increase visibility
- **costly_non_converting** - negative or reduce bid
- **irrelevant_or_low_match** - negative

## Output
Return:
1. harvest keywords
2. negative keyword candidates
3. title-worthy terms
4. bullet-worthy terms
5. advertising notes
