# Traffic Conversion Analyzer

Use this skill to understand whether the problem is traffic, conversion, or both.

## Questions
- Is traffic down?
- Is conversion down?
- Is traffic up but conversion down?
- Is ad traffic replacing organic traffic?
- Is page quality likely involved?

## Output
Return:
1. issue_type
2. evidence
3. likely causes
4. best next action
