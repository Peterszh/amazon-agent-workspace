# 关键词更新规划器

## 核心职责

- 制定周期性词库更新计划
- 识别需要淘汰的词
- 发现新兴趋势词
- 管理季节性关键词

## 更新节奏

### 日常维护（每周）

- 新增高排名新词
- 标记异常词待观察
- 更新词的价值等级

### 月度优化（每月）

- 淘汰低价值词（D级）
- 补充潜力词
- 更新趋势标记

### 季度洗牌（每季度）

- 全面评估词库健康度
- 替换季节性词汇
- 调整词位策略

## 淘汰机制

### 触发淘汰的条件

| 条件 | 状态 | 动作 |
|------|------|------|
| 连续4周D级 | 自动淘汰 | 移出ST |
| 排名持续下滑>20位 | 观察→淘汰 | 替换 |
| 季节过季 | 季节词 | 移出/存档 |
| 转化份额<1%持续2周 | 降级观察 | 优化或淘汰 |

### 淘汰流程

```
识别淘汰候选 → 确认无挽救价值 → 移出ST → 移入观察区
```

## 季节词管理

### 典型季节节点

| 节日/活动 | 时间 | 提前放入 |
|-----------|------|----------|
| Valentine's Day | 2月 | 1月第1周 |
| Easter | 4月 | 3月第1周 |
| Mother's Day | 5月 | 4月第1周 |
| Father's Day | 6月 | 5月第1周 |
| Back to School | 9月 | 8月第1周 |
| Halloween | 10月 | 9月第1周 |
| Black Friday | 11月 | 10月第2周 |
| Christmas | 12月 | 11月第1周 |

### 季节词处理

```
旺季前8周：放入ST，标题预留位置
旺季中：监控转化，及时调整
旺季后1周：移出ST，进入存档
```

## 新词补充

### 来源

1. ABA新晋高排名词
2. 广告Search Term报告中高转化词
3. 竞品Listing挖掘
4. 行业趋势词

### 筛选标准

| 来源 | 价值评估 | 优先级 |
|------|----------|--------|
| ABA新词（排名<50） | 评估后入ST | P1 |
| 广告高CVR词 | 直接入ST | P0 |
| 竞品Listing词 | 验证相关性 | P2 |
| 行业趋势词 | 观察入ST | P3 |
