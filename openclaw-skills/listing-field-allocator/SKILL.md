# 关键词词位分配器

## 核心职责

- 将关键词分配到标题/五点/ST/描述
- 优化关键词覆盖策略
- 检查现有Listing的关键词覆盖缺口

## 分配原则

### 标题（Title）- 最多200字符

**必放词：**
- 核心品类词（流量入口）
- 高价值A/B级词
- 品牌词（自有品牌）
- 核心特性词（差异化卖点）
- 季节旺季词

**字数建议：** 150-180字符
**数量建议：** 3-5个核心词

### 五点（Bullet Points）- 每点最多500字符

**必放词：**
- 场景词（使用场景）
- 特性词（材质、功能）
- 解决痛点词
- 长尾高转化词

**数量建议：** 每点1-2个关键词，自然融入

### Search Terms - 最多250字节

**放置词：**
- 全部相关词（不限数量）
- 标题五点放不下的词
- 低竞争长尾词
- 同义词/拼写变体
- 季节词（旺季前放入）

**注意：** 不要重复标题五点已有的词

### 产品描述（Description）

**放置词：**
- 补充性长尾词
- 技术规格细节
- 售后/认证信息

## 分配决策树

```
关键词 → 价值等级 → 词位

A类词（高流量高转化）
  ├─ 核心品类词 → 标题开头 + 五点 + ST
  ├─ 高转化长尾 → 五点 + ST
  └─ 品牌词 → 标题品牌位

B类词（高转化中等流量）
  ├─ 特性词 → 五点 + ST
  └─ 场景词 → 五点

C类词（中等价值）
  └─ ST填充 + 五点补充

D类词（低价值/观察）
  └─ 暂不投放，持续观察
```

## 输出格式

```json
{
  "关键词": "gaming chair with footrest",
  "价值等级": "A",
  "推荐位置": ["标题", "五点", "ST"],
  "位置优先级": 1,
  "放置示例": {
    "标题": "Ergonomic Gaming Chair with Footrest, Lumbar Support...",
    "五点": "【Ergonomic Design】Features footrest and full lumbar support...",
    "ST": "gaming chair footrest lumbar support ergonomic office"
  },
  "当前状态": "已覆盖|待补充|缺失"
}
```

## 覆盖检查清单

| 字段 | 检查项 | 缺口提示 |
|------|--------|----------|
| 标题 | 核心词1-2个 | ❌ 缺少核心品类词 |
| 五点 | 3-5个特性词 | ❌ 场景词未覆盖 |
| ST | 10-20个相关词 | ⚠️ 长尾词不足 |

## 使用示例

当用户说：
- "这个词应该放标题还是五点"
- "帮我分配这些词的词位"
- "检查我的标题缺什么词"
- "ST应该放哪些词"

调用此skill进行词位分配。
