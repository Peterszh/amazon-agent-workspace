# ABA关键词解析器

## 核心职责

- 解析ABA数据（CSV或飞书Bitable）
- 提取关键词排名、点击份额、转化份额
- 识别异常词（需进一步诊断）
- 词类型初筛：品牌词/泛词/长尾词/季节词

## 输入要求

必须包含字段：
- 搜索排名 (Search Frequency Rank)
- 搜索词 (Search Term)  
- 点击份额 (Click Share)
- 转化份额 (Conversion Share)

## 输出格式

解析后输出结构化数据：

```json
{
  "关键词": "xxx",
  "搜索排名": 1,
  "点击份额%": 13.16,
  "转化份额%": 0.21,
  "词类型": "泛词|品牌词|长尾词|季节词",
  "异常标记": "高排名低点击|高排名低转化|正常|待观察"
}
```

## 词类型识别规则

| 类型 | 识别特征 |
|------|----------|
| 品牌词 | 含品牌名（Nintendo, Lego, Apple等） |
| 泛词 | 单一词根（chair, bottle, headphones） |
| 长尾词 | 3个以上词素+特性词（gaming chair with footrest） |
| 季节词 | 含节日（mothers day, easter, christmas） |

## 异常词标记规则

| 异常类型 | 判定条件 |
|----------|----------|
| 高排名低点击 | 排名<20 且 点击份额<5% |
| 高排名低转化 | 排名<20 且 转化份额<1% |
| 高流量词 | 点击份额>30% |
| 高转化词 | 转化份额>20% |

## 使用示例

当用户说：
- "帮我解析这个ABA文件"
- "从Bitable读取关键词数据"
- "分析这个词为什么排名高但点击低"

调用此skill进行解析，然后交由keyword-value-scorer做价值判断。
