# Product Description Writer

Use this skill for product descriptions.

## Goals
- Explain product value clearly
- Expand on use cases and fit
- Support conversion without sounding exaggerated

## Rules
- Keep the description readable
- Avoid repeating the title and bullets too closely
- Do not add unverified technical claims

## Output
Provide:
1. product description
2. tone note
3. risk note if any data is missing
