# Keyword Library Builder

Use this skill after parsing ABA and ad search term data.

## Build these layers
- **core seed terms** - main category words
- **high-conversion terms** - proven buyers
- **feature terms** - what the product does
- **material terms** - what it's made of
- **audience terms** - who buys it
- **scenario terms** - when it's used
- **long-tail terms** - specific, low competition
- **excluded or risky terms** - avoid or use carefully

## Rules
- Merge duplicates
- Normalize close variants
- Keep strong relevance
- Mark uncertain words

## Output
Return:
1. structured keyword library
2. priority labels
3. where each keyword should be used
