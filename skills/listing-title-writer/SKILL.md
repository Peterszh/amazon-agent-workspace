# Listing Title Writer

Use this skill to draft title options.

## Goals
- Cover core keywords
- Preserve readability
- Reflect product identity and major value
- Match Amazon style expectations for the target marketplace

## Rules
- Do not fabricate specs
- Avoid keyword stuffing
- Avoid absolute or misleading claims
- Prefer natural phrasing

## Output
Provide:
1. balanced version
2. keyword-heavy version
3. conversion-oriented version
With a short note for each.
