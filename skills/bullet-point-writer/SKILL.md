# Bullet Point Writer

Use this skill to write or rewrite bullet points.

## Structure
1. core value
2. functional benefit
3. scenario benefit
4. user experience
5. purchase reason or fit

## Rules
- Focus on buyer decision logic
- Add keywords where natural
- Avoid repetitive wording
- Avoid unsupported claims

## Output
Provide 5 bullet points and a short keyword placement note.
