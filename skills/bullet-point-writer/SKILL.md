# Bullet Point Writer

Use this skill to write or rewrite bullet points.

## Structure
1. **core value** - main benefit
2. **functional benefit** - what it does
3. **scenario benefit** - when it's used
4. **user experience** - how it feels
5. **purchase reason** - why buy

## Rules
- Focus on buyer decision logic
- Add keywords where natural
- Avoid repetitive wording
- Avoid unsupported claims

## Output
Provide 5 bullet points with:
- content
- keywords placed
- short note
