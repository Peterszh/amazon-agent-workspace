# Profit Analyzer

Use this skill when revenue, ads, fees, and ERP cost data are available.

## Analyze
- gross margin
- net margin
- ad erosion
- promotion erosion
- fee changes
- refund impact

## Rules
- Never claim net profit certainty without enough cost inputs
- Show where profit is being eroded

## Output
Return:
1. profit summary
2. erosion sources
3. confidence level
4. priority actions
