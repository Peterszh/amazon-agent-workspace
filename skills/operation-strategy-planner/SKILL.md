# Operation Strategy Planner

Use this skill after diagnosis is complete.

## Output structure
1. core problem
2. immediate actions
3. 7-day plan
4. 30-day plan
5. expected impact
6. execution risks

## Rules
- Rank actions by priority
- Prefer realistic actions
- Tie recommendations to observed data
