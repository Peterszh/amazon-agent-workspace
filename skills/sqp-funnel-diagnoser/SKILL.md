# SQP漏斗诊断器

## 核心职责

- 诊断搜索-点击-加购-购买全链路问题
- 识别关键词在漏斗哪个环节流失
- 对应到Listing具体问题（标题/五点/价格/评价）
- 给出优化方向

## 漏斗模型

```
展示(Impression) → 点击(Click) → 加购(Add to Cart) → 购买(Purchase)
     ↓                ↓              ↓               ↓
  搜索排名        标题/主图        五点/详情        价格/评价
  关键词选品      价格/评分        卖点展示          促销力度
```

## 问题诊断矩阵

### 问题1：高展示低点击

**漏斗阶段：** 搜索 → 点击
**表现：** 排名高但点击率低

| 可能原因 | 诊断要点 | 优化方向 |
|----------|----------|----------|
| 标题问题 | 核心词不突出 | 标题前移核心卖点 |
| 主图问题 | 主图不吸引 | 主图优化 |
| 价格问题 | 价格高于均价 | 价格调整 |
| 评分问题 | 评分低于4.0 | 评价管理 |
| BSR排名 | 低于大类目前20% | 提升BSR |

**关键词特征：**
- 排名<50，点击份额<5%
- 标题五点未覆盖该词

### 问题2：高点击低加购

**漏斗阶段：** 点击 → 加购
**表现：** 有流量但留不住

| 可能原因 | 诊断要点 | 优化方向 |
|----------|----------|----------|
| 五点问题 | 卖点不突出 | 五点重写 |
| 详情问题 | 细节不足 | A+优化 |
| 图片问题 | 场景图少 | 图片补充 |
| 规格问题 | 尺寸/特性不明 | 补充规格 |

**关键词特征：**
- 点击份额5-15%，转化份额<3%
- 五点未深度覆盖该词场景

### 问题3：高加购低购买

**漏斗阶段：** 加购 → 购买
**表现：** 有兴趣但不下单

| 可能原因 | 诊断要点 | 优化方向 |
|----------|----------|----------|
| 价格问题 | 高于竞品 | 价格策略 |
| 评价问题 | 差评/数量少 | 评价管理 |
| 竞品问题 | 竞品更优 | 差异化定位 |
| 促销问题 | 无促销刺激 | 促销设置 |

**关键词特征：**
- 转化份额高（>10%）但实际订单少
- 可能被竞品截胡

## 关键词-问题映射

```
品类核心词（gaming chair）
  └─ 高展示低点击 → 标题不够吸引 → 优化标题核心词位置

特性词（ergonomic with footrest）
  └─ 高点击低加购 → 五点未突出该特性 → 五点强调特性

长尾词（gaming chair for tall person）
  └─ 高加购低购买 → 规格/尺寸不匹配 → 补充详细规格

季节词（mothers day gift）
  └─ 整体转化低 → 竞品价格/评价优势 → 调整定价策略
```

