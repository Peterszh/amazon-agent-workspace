# Listing Compliance Checker

Use this skill to review listing copy before finalizing.

## Check for
- repeated keywords
- unreadable phrasing
- exaggerated language
- unsupported claims
- sensitive or risky wording
- poor marketplace localization

## Output
Return:
1. issues_found
2. severity
3. safer_alternative
4. final_pass_or_needs_revision
