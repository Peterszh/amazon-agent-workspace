# 关键词价值评估器

## 核心职责

- 判断词值不值得用
- 识别词的价值类型
- 分类为：核心词/潜力词/观察词/淘汰词
- 结合CVR和流量做综合评分

## 评估维度

### 1. 流量价值矩阵

| 点击份额 | 转化份额 | 价值判断 |
|----------|----------|----------|
| >30% | >20% | ⭐高价值核心词 |
| >30% | <5% | 高流量低转化-观察 |
| 10-30% | >15% | 高转化潜力词 |
| 10-30% | 5-15% | 中等价值词 |
| <10% | >10% | 长尾高转化词 |
| <5% | <5% | 低价值淘汰候选 |

### 2. 词类型价值排序

```
第一梯队（必争）：
  - 核心品类词（gaming chair）
  - 高转化长尾词（gaming chair with footrest ergonomic）

第二梯队（提升）：
  - 潜力词（点击份额10-30%）
  - 场景词（office gaming chair）

第三梯队（观察）：
  - 季节词（mothers day gift）
  - 新兴词（排名上升但不稳定）

第四梯队（淘汰/低优）：
  - 高排名低点击词
  - 品牌词（除非自有品牌）
  - 泛词（竞争激烈）
```

### 3. 评分公式

```
综合得分 = 流量权重×点击份额 + 转化权重×转化份额

流量权重建议：0.4
转化权重建议：0.6（转化更重要）
```

## 输出格式

```json
{
  "关键词": "gaming chair with footrest",
  "搜索排名": 50,
  "点击份额%": 8.5,
  "转化份额%": 12.3,
  "价值评分": 10.78,
  "价值等级": "A|B|C|D",
  "词分类": "核心词|潜力词|观察词|淘汰词",
  "建议动作": "立即投放|优化页面|持续观察|移出ST"
}
```

## 价值等级定义

| 等级 | 定义 | 建议动作 |
|------|------|----------|
| A | 高流量高转化 | 标题+五点+ST必填，广告必抢 |
| B | 高转化中等流量 | 五点+ST优先，广告提升 |
| C | 中等价值 | ST填充，持续观察 |
| D | 低价值 | 观察或淘汰 |

## 使用示例

当用户说：
- "这个词值不值得用"
- "帮我判断这个词应该打广告吗"
- "这个词是高价值词吗"
- "这个词属于核心词还是长尾词"

调用此skill进行价值评估。
