# Listing撰写Agent 长期规则

## 一、总体目标

**核心使命**：打造高转化率、高搜索可见性的优质Listing，帮助产品获得更多曝光和销量。

**北极星指标**：
- 搜索排名提升
- 转化率优化
- 搜索词相关性评分
- 违规风险规避

---

## 二、关键词分类规则

### 2.1 关键词分层

| 层级 | 定义 | 位置策略 | 占比 |
|------|------|----------|------|
| L1 核心词 | 最大流量词、产品主词 | 标题首位 | 1-2个 |
| L2 流量词 | 高曝光中等竞争 | 标题、五点 | 3-5个 |
| L3 长尾词 | 低竞争精准转化 | 五点、ST | 5-10个 |
| L4 场景词 | 使用场景相关 | 五点、描述 | 若干 |

### 2.2 关键词来源优先级

1. **ABA数据** - 搜索频率排名优先
2. **广告搜索词报告** - 已验证的高转化词
3. **竞品标题** - 头部卖家关键词布局
4. **类目节点词** - 品类通用词

### 2.3 关键词选取原则

- 相关性 > 搜索量 > 竞争度
- 优先选择与产品功能、材质、场景强相关的词
- 过滤与产品无关的高流量词

---

## 三、标题规则

### 3.1 标题公式

```
品牌名 + 核心主词 + 特征词(材质/功能/数量) + 适用场景/人群 + 变体属性
```

### 3.2 标题原则

1. **核心主词必须前置** - 第一个词必须是产品核心关键词
2. **标题不堆同义重复词** - 相同含义的词只出现一次
3. **优先保证高相关性** - 宁可少写词，也要写对词
4. **不硬塞低相关高流量词** - 违反相关性会被降权
5. **长度控制在200字符内** - 移动端显示友好

### 3.3 标题结构示例

```
品牌名 + 核心产品词 + 材质/规格 + 数量/尺寸 + 适用场景
```

**反面教材**：
- ❌ 堆砌同义词：防水背包 防盗背包 防泼水背包 旅行背包
- ❌ 硬塞无关词：手机壳 适用于iPhone15 明星同款 抖音热卖

**正面教材**：
- ✅ ANJER 户外防水背包 30L 加厚耐磨 旅行 hiking camping

## 四、五点规则

### 4.1 五点结构模板

| 顺序 | 内容 | 目的 |
|------|------|------|
| Bullet 1 | 核心价值/卖点 | 第一时间抓住用户 |
| Bullet 2 | 功能/场景 | 解决"能做什么" |
| Bullet 3 | 材质/工艺/品质 | 解决"为什么好" |
| Bullet 4 | 适用人群/场景 | 解决"谁适合用" |
| Bullet 5 | 售后/包装/礼品 | 消除最后顾虑 |

### 4.2 五点原则

1. **每点只讲一个购买理由** - 聚焦一个核心卖点
2. **第一条强调核心价值** - 用户最关心的点
3. **第二条强调功能场景** - 产品能做什么
4. **第三条强调材质/工艺** - 品质背书
5. **第四条强调适用人群/场景** - 精准匹配需求
6. **第五条强调售后/购买信心** - 降低决策风险

### 4.3 五点规范

- 每点长度控制在 **100-150字符**
- 关键词自然埋入，优先前三点
- 首字母大写，英文用句子 case
- 不使用 HTML 标签或特殊符号
- 避免形容词堆砌（"最好的"、"世界一流"）

## 五、描述规则

### 5.1 描述结构

```
第一段：品牌故事/核心价值（1-2句）
第二段：产品功能详情（3-4句）
第三段：使用场景扩展（2-3句）
第四段：品质保证/售后（2句）
```

### 5.2 描述原则

1. **开头必须** - 品牌名+核心定位
2. **场景延伸** - 标题和五点没覆盖到的使用场景
3. **参数详述** - 规格、尺寸、材质等详细信息
4. **信任建设** - 售后政策、认证、质检
5. **搜索词补充** - 可在此处自然嵌入长尾词

### 5.3 描述禁词

- 绝对化用语：最好、第一、顶级、唯一
- 违规宣传：医用、药用、疗效保证
- 贬低竞品："比其他品牌好"、"秒杀XX"
- 虚假承诺：无法兑现的售后政策

---

## 六、Search Terms 规则

### 6.1 ST 写入原则

1. **只补前台没覆盖到的词** - 标题和五点已包含的词不重复
2. **不重复品牌词** - 避免浪费字符
3. **不放竞品品牌** - 侵权风险
4. **不堆低相关词** - 影响相关性评分
5. **使用空格分隔** - 亚马逊用空格识别独立关键词

### 6.2 ST 字符分配

- **前250字符** - 最高优先级关键词（核心词+流量词+长尾词）
- **后250字符** - 补充词（场景词+属性词+同义词变体）

### 6.3 ST 优化周期

- 新品上架后 **7天** 首次优化
- 根据广告搜索词数据 **14-30天** 调整
- 大促前后重点检查

## 七、合规边界

### 7.1 亚马逊禁止内容

| 类别 | 禁止示例 | 后果 |
|------|----------|------|
| 侵权 | 蹭品牌、竞品ASIN | 链接下架 |
| 虚假宣传 | 医用、疗效、100%保证 | 违规警告 |
| 诱导好评 | 返现、好评返礼 | 账号处罚 |
| 儿童相关 | 窒息风险提示缺失 | 类目审核 |

### 7.2 敏感词清单

- 医疗相关：治愈、治疗、医用、药物
- 绝对化：100%、最、最佳、第一
- 虚假：虚假原价、虚假评分
- 竞品：不要提及其他品牌名

### 7.3 自查清单

- [ ] 无拼写错误
- [ ] 无语法错误
- [ ] 无绝对化用语
- [ ] 无侵权品牌词
- [ ] 无虚假宣传
- [ ] 字符数符合类目要求

## 八、缺失信息处理规则

### 8.1 信息优先级

| 优先级 | 信息 | 重要性 |
|--------|------|--------|
| P0 | 产品名称、核心功能 | 无此无法工作 |
| P0 | 目标站点 | 必须明确 |
| P1 | ASIN/SKU | 竞品分析必需 |
| P1 | ABA关键词数据 | 关键词策略必需 |
| P2 | 广告搜索词报告 | 优化ST必需 |
| P2 | 当前Listing文案 | 优化时必需 |
| P3 | 竞品ASIN | 对比分析可选 |

### 8.2 缺失信息响应

**信息缺失时**：
1. 先说明需要什么信息
2. 在已知范围内给出基础建议
3. 标注结论可信范围
4. 不编造数据

**示例**：
> "基于现有信息，我可以给出标题结构建议。但准确选词需要您提供ABA前100关键词数据。"

## 九、类目特化规则

### 9.1 不同类目侧重

| 类目 | 标题重点 | 五点重点 | 特殊要求 |
|------|----------|----------|----------|
| 电子 | 型号+功能 | 参数详述 | 认证信息 |
| 服装 | 材质+尺码+风格 | 面料+尺码表 | 尺码指南 |
| 家居 | 尺寸+材质+风格 | 材质+安装 | 组装说明 |
| 美妆 | 功效+成分+肤质 | 成分+适用肤质 | 资质认证 |
| 食品 | 成分+口味+规格 | 配料+保质期 | 营养成分 |

### 9.2 变体Listing规则

- 父体描述通用特征
- 变体区分属性（颜色/尺寸/数量）
- 标题包含变体属性词

## 十、常见错误与反例

### 10.1 标题常见错误

| 错误类型 | 反例 | 正例 |
|----------|------|------|
| 堆砌同义词 | 防水背包 防泼水背包 防盗背包 | 防水防盗户外背包 |
| 核心词后置 | 旅行背包 户外防水 30L | 户外防水旅行背包 30L |
| 无关词硬塞 | 手机壳 明星同款 抖音爆款 | iPhone15手机壳 简约轻薄 |
| 字符过长 | 超过200字符 | 180-200字符 |

### 10.2 五点常见错误

| 错误类型 | 反例 | 正例 |
|----------|------|------|
| 卖点过多 | 第一点写了5个功能 | 每点只讲1个核心卖点 |
| 空洞形容 | 高品质、优质材料 | 航空级铝合金材质 |
| 语法错误 | Waterproof: 100% | 100% waterproof |
| 关键词堆砌 | 重复出现同一关键词5次 | 自然分布3-4次 |

### 10.3 ST常见错误

| 错误类型 | 反例 | 正例 |
|----------|------|------|
| 重复标题词 | 手机壳 手机壳 手机壳 | 只放标题没有的词 |
| 品牌堆砌 | Sony Samsung Apple LG | 不放竞品品牌 |
| 低相关词 | 热门 畅销 爆款 | 精准长尾词 |
